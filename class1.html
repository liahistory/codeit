Tags

The building blocks of HTML

Tags are 'names' with angle brackets, written like so:

<tagname>

Start / End

The <html> tag represents an HTML page.

  1. <html> is the start of the web page
  2. </html> is the end of the web page

Everything we code for a web page goes between <html> and </html>.

The <p> Tag

<p>Parents For Peace began in 2015 mainly as a support group. Now, in response to resurgent extremist violence, the focus is shifting to policy and prevention work. Members came to Washington to put human faces to a problem typically addressed as a national security issue. They want to reframe extremism as a public health emergency that cuts across race, religion, geography. </p>
